# Hero

## 1. Introduction

This document provides an overview of the hero unit in **Pillage First, Ask Questions Later**. It
covers implementation details, notes important files, and caveats.

## 2. Overview

Each player has a special **Hero** unit. This unit is unique, and only one per player is permitted. Hero is the
strongest unit in the game, and is also the only unit in the game that brings special bonuses to the village it's
currently in.
